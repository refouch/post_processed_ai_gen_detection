### Data collection and processing

The dataset must be downloaded manually from this website: https://zenodo.org/records/14963880

ONLY download the 20GB ```RRDataset_test.tar.gz``` acrhive, the other one is irrelevant to us, and unpack it as is in this folder

If needed, run the ```data_processing.py``` script in order to produce the csv linnking each file path to a specific balanced train/test/split
